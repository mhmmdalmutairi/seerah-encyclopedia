📡 واجهة برمجة التطبيقات (API)

موسوعة السيرة النبوية المؤسسية تتيح بياناتها مجاناً ومفتوحاً للباحثين والمطوّرين. لا تحتاج مفتاحاً، لا تسجيلاً، لا حدّ معدّل.

عام CORS مفعّل CC BY 4.0

1. النقاط النهائية (Endpoints)

GET /data.json

قاعدة بيانات الكيانات الكاملة (~1.2MB).

https://mhmmdalmutairi.github.io/seerah-encyclopedia/data.json

GET /sources/manifest.json

قائمة الجولات البحثية ومجموع الكيانات لكل جولة.

https://mhmmdalmutairi.github.io/seerah-encyclopedia/sources/manifest.json

GET /schema.json

JSON Schema يصف بنية الكيانات (الحقول المتاحة والقيم الممكنة).

https://mhmmdalmutairi.github.io/seerah-encyclopedia/schema.json

2. مثال استخدام (JavaScript)

// تحميل كل الكيانات
const res = await fetch("https://mhmmdalmutairi.github.io/seerah-encyclopedia/data.json");
const { entities } = await res.json();

// فلترة الكيانات الإندونيسية
const indonesian = entities.filter(e => e.country === "ID");
console.log(`${indonesian.length} كياناً إندونيسياً`);

// كيانات السيرة الرقمية
const digital = entities.filter(e => e.subjects?.includes("digital_seerah"));

// كيانات تأسّست بعد 2010
const recent = entities.filter(e => e.founded && e.founded >= 2010);

3. مثال (Python)

import requests

r = requests.get("https://mhmmdalmutairi.github.io/seerah-encyclopedia/data.json")
data = r.json()
entities = data["entities"]

# المؤسسات الصوفية
sufi = [e for e in entities if "sufism_seerah" in (e.get("subjects") or [])]
print(f"{len(sufi)} مؤسسة صوفية")

# توزيع حسب الدولة
from collections import Counter
by_country = Counter(e.get("country") for e in entities)
print(by_country.most_common(10))

4. الحقول الرئيسة

الحقلالنوعالوصف
idstringمعرّف فريد (مثل NAM-104)
name_arstringالاسم بالعربية
name_enstringالاسم بالإنجليزية (إن وُجد)
typestringنوع المؤسسة (28 نوعاً)
type_groupstringالمجموعة الرئيسة (6 مجموعات)
regionstringالإقليم الجيوبوليتيكي (7 أقاليم)
countrystringكود ISO 3166-1 alpha-2
foundedintegerسنة التأسيس
description_arstringوصف بالعربية
urlstringالموقع الرسمي
subjectsarrayالموضوعات السيرية المخدومة
subjects_categoryarrayتصنيفات الموضوعات الكبرى
key_figuresarrayالشخصيات الرئيسة
verificationstringدرجة التحقّق: field_verified, desk_verified, needs_verification
funding_typestringنوع التمويل: state, religious_authority, private, academic_host, waqf, mixed
statusstringالحالة: active, dormant, suspended, closed
roundstringالجولة البحثية التي أُضيف فيها
